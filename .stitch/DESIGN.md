# Smartaf Mobile Design System

## Overview
- Product mood: calm, editorial, trustworthy, and Hebrew-first.
- Core visual idea: light layered surfaces, rounded shapes, soft gradients, and spacious RTL composition.
- UX priority: make childcare feel safe and premium, not transactional or crowded.

## Color System
- Base canvas: `#f4f6ff`
- Section surface: `#ecf1ff`
- Interactive recessed surface: `#d5e3ff`
- Elevated white card: `#ffffff`
- Primary gradient: `#702ae1` -> `#6411d5`
- Primary accent surface: `#b28cff`
- Secondary muted surface: `#e9def5`
- Main text: `#242f41`
- Secondary text: `#515c70`
- Divider / outline: `#a2adc4`

## Typography
- Headlines: `Plus Jakarta Sans`, usually `700` to `800`
- Body and labels: `Be Vietnam Pro`
- Screen titles: `30 / 38`
- Top bar titles: `22 / 30`
- Card titles: `18 / 26`
- Body large: `16 / 24`
- Body: `15 / 22`
- Caption: `12 / 18`
- RTL rule: align text right by default and let whitespace create calm hierarchy.

## Spacing And Shape
- Main screen padding: `24`
- Card padding: `20` to `24`
- Large section gaps: `24` to `32`
- Control height: `56` to `64`
- Cards: `20` to `24` radius
- Hero surfaces: `24` to `32` radius
- Chips and CTA pills: fully rounded or large rounded rectangles

## Elevation
- Prefer tonal layering over visible borders.
- Use diffused shadows only for hero buttons, floating cards, and focal icons.
- When a border is needed, keep it subtle and low-contrast.

## Component Guidance
- Welcome / auth screens should have a light blurred top bar with brand treatment.
- Primary CTA buttons should use the Smartaf purple gradient and feel soft, not sharp.
- Secondary CTA buttons should use muted purple surfaces without heavy borders.
- Input fields should feel recessed inside the layout, using the blue-tinted surface color.
- Role cards should be large editorial blocks with an icon circle, right-aligned copy, and a clear selected state.
- OTP entry should use separate visual boxes with strong focus feedback and generous spacing.

## Stitch Prompt Notes
- Use phrases like `editorial mobile onboarding`, `Hebrew RTL`, `soft trust-building surfaces`, `purple gradient CTA`, and `glassmorphism top bar`.
- Avoid dense dashboard layouts for auth and onboarding steps.
- Prefer asymmetric glow shapes and soft circles over hard dividers.
- Keep footer and support content quiet and low-contrast.
